## ✨ Features

### 🔑 User Authentication
- Secure **Sign up & Log in** with **JWT (JSON Web Tokens)**.  
- Passwords are hashed with **bcrypt** before storing in the database.  

### 🏠 Property Listings
- Browse houses, villas, and unique accommodations.  
- **Filter** by location, amenities, price, and availability.  

### 📅 Booking System
- Book properties for specific dates with a built-in calendar.  
- View booking history & upcoming reservations.  

### 👨‍💼 Host Management
- Hosts can list their own properties.  
- Manage property details, pricing, and availability.  

### 📱 Responsive Design
- Built with **Material UI** for a clean, user-friendly interface.  
- Fully responsive across desktop & mobile devices.  

---

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js  
- 📦 Redux (State Management)  
- 🎨 Material UI + Sass (SCSS)  

### Backend
- 🟢 Node.js + Express.js  
- 🍃 MongoDB + Mongoose (ODM)  
- 🔐 JWT + Bcrypt (Authentication & Security)  

---

## 🔑 Key Features in Detail
- Sign up & Log in with JWT Authentication  
- Create new Listings with property details  
- Upload, delete, drag & drop property photos  
- Calendar-based booking system  
- Wishlist functionality (add & remove)  
- Property feed by category  
- Keyword-based property search  
- Local photo uploads stored in MongoDB  
- Structured Mongoose Schemas  
- Centralized state management with Redux
